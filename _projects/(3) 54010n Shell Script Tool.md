# 54010N Tool
The basic bash script tool - Made by 54010N Team!
![54010N Tool](https://raw.githubusercontent.com/nh4ttruong/54010nTool/main/thumb.jpg)

## Demo Tool
You can view the demo video [here](https://youtu.be/Aekr2aW-kps)

## Functions
- File Management
  - Create/Delete file
  - Set permission
  - Move/Copy file
- User Management
  - Create new user
  - Lock user
  - Backup user folder
  - Show all user
- Disk Checking & Email Alert
  - Backup any folder
  - Auto alert to email
- Processes Management
  - Show all process
  - Show detail process by PID
  - Terminate a process
  - Show RAM info
- Stuff
  - whois tool interface
  - Calculator
  - Archive backup file and Email
